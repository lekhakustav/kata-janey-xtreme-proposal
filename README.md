# Kata Janey x Xtreme Proposal

Static proposal site prepared for GitHub Pages.

Files:
- `index.html`
- `style.css`
- `script.js`
- `assets/`

GitHub Pages steps:

1. Create GitHub repo
2. Upload files
3. Go Settings -> Pages
4. Source: Deploy from branch
5. Branch: `main`, folder: `/root`
6. Save

Final link format:

[https://yourusername.github.io/repo-name/](https://yourusername.github.io/repo-name/)
